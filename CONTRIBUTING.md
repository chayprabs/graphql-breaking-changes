# Contributing

Thanks for your interest in GraphQLGuard!

1. Fork the repository and create a branch from `main`.
2. Install dependencies: `pnpm install`
3. Make changes in `packages/core` and/or `packages/web`.
4. Run checks: `pnpm typecheck && pnpm test && pnpm build`
5. Open a pull request with a clear description.

Please keep browser-only privacy guarantees: do not add analytics that transmit schema contents.
